# 🎬 Netflix Data Analysis

## 📌 Overview
This project involves loading, cleaning, analyzing and visualizing 
data from the Netflix dataset using Python. The goal is to explore 
content trends and derive meaningful insights.

## 🛠️ Libraries Used
- **Pandas** - Data loading and cleaning
- **Matplotlib** - Data visualization
- **Seaborn** - Enhanced visualizations

## 📂 Dataset
- File: `netflix_titles.csv`
- Total entries: 8807
- Columns: show_id, type, title, director, cast, country, 
  date_added, release_year, rating, duration, listed_in, description

## 🔍 Data Cleaning
- Removed duplicate rows
- Filled missing values in director, cast, country
- Converted date_added to datetime format
- Removed unnamed columns

## 📊 Visualizations & Key Insights
- 🎬 Netflix has **6131 Movies** vs **2666 TV Shows**
- 🎭 Top genres: International Movies, Dramas, Comedies
- 🎬 Most movies are **90-100 minutes** long
- 📺 Most TV shows have only **1 season**
- 📈 Netflix content grew massively until 2019
